# Projeto 12 - App Help Desk

## Sobre o app

O objetivo deste projeto foi praticar os princípios básicos da programação backend com php.


## Melhorias que poderiam ser implementadas:
* Este sistema não é seguro;
* Uso de um banco de dados
* Função de registrar usuários
* Permissão para o adm remover usuário
* Permissão para o adm administrar e responder os chamados
* etc...

## Como executar
Mova os arquivos deste projeto para seu servidor Apache.
